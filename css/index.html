TECH::CAMPとは

ABOUT

プログラミングを学習するために最高の環境を提供しています。

最高のテキスト

市販の参考書やフリーの動画だと、「その時点では覚える必要のない内容」が多く含まれています。本キャンプのオリジナル学習システムは本当に重要な内容だけを抽出し、効率的にステップアップできるように最大限工夫されています。テキストの詳しい紹介はこちら

実践型学習コンテンツ

TECH::CAMPは、いわゆる文法だけを学ぶのではなく実際にサービスを作りながら実践的に学んでいきます。「分かる」から「出来る」に変わるには、同じ内容でも手が自然と動くまで反復することが重要です。TECH::CAMPでは、とにかく手を動かして体で覚えるまで実践を積み重ねていきます。

質問し放題の環境

学習効率をあげるには、1人で悩んでいる時間をいかに減らすかが重要です。キャンプに直接来ていただいている方はもちろん、オンラインでも常に待機しているので素早く回答します。（対応時間:土日を含む11時〜23時/通話+画面共有できる環境でのみ対応）

よくあるご質問

FAQ

入金はどのタイミングで行えばよいですか？

事前課題と面談による選考の結果、合格した場合にメールにて入金のお知らせを致します。その際、口座振込みかクレジットカード決済かをお選びいただき指定の期日までに入金をお願いします。

選考の結果はいつ分かりますか？

事前課題の提出後、面談の設定を致します。面談終了後、3日以内にメールにて選考の結果をお伝えさせていただきます。

事前課題は未経験者でも解けますか？

事前課題を解く際に、基礎知識を学ぶための教材をお渡しします。そのため、未経験者でも教材を学習することで課題を解くことができます。

選考で落ちてしまった場合はどうなりますか？

選抜で落ちてしまった場合は残念ですが、次の期以降に再度受講の申し込みをお願いします。一度落ちてしまっても、次回以降の期に申し込むことは可能です。

次回以降の期はいつ開催ですか？

TECH::CAMP EXPERTは1ヶ月ごとに開催予定です。開催日が確定いたしましたら、本サイト内に掲載いたしますのでご確認お願いします。

月々いくらで分割で支払いができますか？

分割はクレジットカード決済のみとなるので、お使いのカード会社の分割回数によって価格が変動します。ほとんどのカードでは最大24回、もしくは36回の分割が可能です。

事前に学習は必要ですか？

事前課題を解くためにRubyの基本的な知識を学習する必要があります（参考となる教材をお渡しします）。それ以外の知識に関しては2週間のTECH::CAMP通常コースで学習を行うので事前学習は必要ありません。
